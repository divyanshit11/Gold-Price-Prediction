# Gold Price Prediction

A machine learning project focused on analyzing and predicting gold prices using historical financial data.

## What it does

* Cleans and preprocesses gold price dataset
* Performs exploratory data analysis using histograms and heatmaps
* Visualizes gold price trends over time
* Trains machine learning models for prediction
* Evaluates model performance using R² score

## Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

## Conclusion

* Feature analysis and trend visualization helped understand gold price movements
* Random Forest and regression models achieved strong prediction performance
* Proper preprocessing and feature importance analysis improved model accuracy
